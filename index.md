### [Аналитика клиентской удовлетворённости в travel-индустрии](https://github.com/pnppl1/apr25_bde_int_supply)
**Цель:** Создать end-to-end систему для сбора, хранения и анализа клиентских отзывов компаний в туристической отрасли, чтобы обеспечить объективную оценку удовлетворённости клиентов.
**Инструменты:** Beautiful Soup, PostgreSQL, FastAPI, Vader, TextBlob, BERT, Dash, Docker
**Результат:** End-to-end продукт: централизованная БД отзывов, sentiment-анализ, API и интерактивный дашборд, контейнеризация и готовность к масштабированию.
**Моя роль:** участие в полном цикле проекта: сбор данных → моделирование → анализ → API → деплой
### [Анализ поисковых запросов: Яндекс Картинки]
**Цель:** Проанализировать поисковые запросы сервиса Яндекс Картинки и выявить различия в пользовательском поведении на платформах Desktop и Touch в рамках тестового задания.
**Инструменты:** Python, pandas, matplotlib, pandasql (SQL), Jupyter Notebook, Yandex DataLens.
**Результат:** Проведён полный цикл анализа данных и сравнение платформ Desktop и Touch; выявлены различия в пользовательских паттернах, создан интерактивный дашборд с KPI и визуализациями.
